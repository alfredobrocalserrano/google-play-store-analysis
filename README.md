# 📱 Análisis Estratégico de la Google Play Store

Este proyecto analiza más de 10.000 apps de la Play Store para identificar los factores que determinan el éxito (ratings e instalaciones).

### 📂 Origen de los Datos
Los datos utilizados en este análisis provienen del dataset público de Kaggle: 
[Google Play Store Apps](https://www.kaggle.com/datasets/gauthamp10/google-playstore-apps).

*El dataset original contiene información sobre más de 10,000 aplicaciones, incluyendo categorías, ratings, tamaño y precios.*

## 🛠️ Tecnologías Utilizadas
* **Python (Google Colab):** Limpieza de datos, gestión de nulos y normalización de unidades (MB/KB).
* **Power BI:** Creación de un dashboard interactivo de 4 páginas.
* **DAX:** Creación de medidas personalizadas y segmentos de tamaño.

## 📊 Estructura del Dashboard
1. **General:** Visión global del mercado y categorías líderes.
2. **Precios y Calidad:** El impacto del modelo de pago vs gratuito en el rating.
3. **Tamaño y Rendimiento:** Cómo el peso de la app afecta a las instalaciones.
4. **Perfil de Éxito:** Recomendaciones finales para lanzar una app.

## 📈 Conclusiones Clave
* Las apps de **pago** tienen un rating medio **0.49 puntos superior** a las gratuitas.
* Existe una barrera psicológica de descarga a partir de los **100MB**.
* El "Punto Dulce" para el éxito masivo es una app **gratuita de menos de 100MB**.
